PHPStats::Matrix::__construct ( $  literal)

Constructor function

Creates a new matrix object. If given a pair of numbers, this will return a zero-filled matrix with ones running down the diagonal, i.e. an identity matrix if the matrix is square. If given a string like '[1, 2, 3; 4, 5, 6; 7, 8, 9]' as the first argument, then it will attempt to construct a matrix using the supplied values.

Parameters:
string$ A string representing a matrix literal
Returns:
matrix A new matrix object.
